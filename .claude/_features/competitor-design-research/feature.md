## Description

Use Firecrawl to systematically scrape and analyze competitor health apps and related products, capturing their color palettes, typography, layout patterns, and UX flows. The gathered insights will be documented as a design reference so the team can make informed decisions when building Journey Health Navigator's own visual identity. This is a research task that produces structured design notes, not shipped code.

## Key Points

- Crawl 3–6 competitor or adjacent health/wellness app websites and landing pages using Firecrawl
- Extract and document color palettes, fonts, spacing conventions, and notable UI patterns
- Produce a design-reference artifact (Markdown notes + color swatches) stored in the repo for the team to reference during implementation
